# Client Evaluation Project

## Setup / Usage
This repository contains:
- Prompt Template
- Custom Metrics Definitions
- Evaluation Dataset (CSV/JSON)
- Final Report (PDF)

## Files
- /data/evaluation.csv
- /data/evaluation.json
- prompt_template.txt
- metrics.json
- report.pdf

## How to use
1. Download files
2. Upload to GitHub repository via web interface:
   - Go to https://github.com/iamjunaidrafique
   - Click "New Repository" (or open existing repo)
   - Click "Add file → Upload files"
   - Drag & drop all files
   - Commit changes

No VS Code required.
